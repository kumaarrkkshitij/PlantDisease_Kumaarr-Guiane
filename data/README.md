# Dataset Instructions

## Dataset
- PlantVillage dataset (38 plant classes, ~43,444 images)
- Source: https://www.kaggle.com/datasets/mohitsingh1804/plantvillage

## Scripts
- `get_data.py` → downloads and organizes dataset
- `download.sh` → alternative shell script to fetch dataset

## Notes
- Ensure no corrupted or blurred images.
- Organize dataset in train/validation folders for model training.
